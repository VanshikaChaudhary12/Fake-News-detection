How It Works

Understanding our AI-powered detection process

1

Input Text

User submits news article or social media content for analysis through our web interface.

2

Text Preprocessing

Content is cleaned by removing URLs, special characters, and applying lemmatization with stopword removal.

3

Feature Extraction

Text is converted to numerical features using TF-IDF vectorization with 10,000 features and 1-3 gram analysis.

4

AI Classification

Enhanced Logistic Regression model analyzes features and predicts authenticity with confidence score.

5

Results Display

System displays classification (Fake/Real), confidence percentage, and detailed technical metrics.

Technical Architecture

Dataset

45,000+ articles from Kaggle Fake News Dataset

Algorithm

Logistic Regression with hyperparameter tuning

Features

TF-IDF with 10K features, 1-3 grams

Accuracy

94%+ detection accuracy rate

TF-IDF Formula

TF-IDF(t,d) = TF(t,d) × IDF(t)

Where:

  • TF(t,d) = (Number of times term t appears in document d) / (Total terms in document d)
  • IDF(t) = log(Total documents / Documents containing term t)
